ARCHITECTURE OVERVIEW
================================================================================

CORE CLASSES:

  1. PolymarketAPI
     - Fetches live market data from Gamma API
     - Categorizes markets via regex pattern matching
     - Extracts price and probability data
     - Handles retries and timeouts

  2. FinancialMarketSimulator
     - Provides mock financial market prices
     - Returns implied probabilities for each instrument
     - Production: Replace with live market data feeds

  3. ArbitrageDetector
     - Main detection engine
     - Processes each market for opportunities
     - Applies filtering and Kelly sizing
     - Generates trade recommendations
     - Formats output and exports JSON

  4. ArbitrageOpportunity (dataclass)
     - Immutable data container
     - Stores all opportunity metadata
     - Serializable to JSON
     - Type-safe with annotations

DATA FLOW:

  Gamma API → Categorize → Extract Prices → Financial Lookup
       ↓
  Compare Probabilities → Filter (>2%) → Size with Kelly
       ↓
  Generate Trades → Sort by Edge → Format Output → Export JSON
