# Lumina Pulse Insights - Static Blog Site

A fully optimized static blog site for "Lumina Pulse Insights" featuring expert guides on AI trading, prediction markets, and passive income automation. Built for free GitHub Pages hosting with built-in SEO, AdSense integration, and affiliate monetization.

## Quick Start (5 Minutes)

### Step 1: Fork/Clone to GitHub
1. Create a new GitHub repository named `yourusername.github.io`
2. Clone this repository into it or copy files directly
3. Commit and push to GitHub

### Step 2: Enable GitHub Pages
1. Go to repository Settings → Pages
2. Select "Deploy from a branch"
3. Choose `main` branch, `/root` folder
4. Save - your site goes live at `yourusername.github.io`

### Step 3: Add Your Custom Domain (Optional)
1. Buy a domain from Namecheap, GoDaddy, or similar
2. Edit `CNAME` file and replace with your domain: `yourdomain.com`
3. Configure DNS at your registrar:
   - Add A records pointing to GitHub Pages IPs:
     - `185.199.108.153`
     - `185.199.109.153`
     - `185.199.110.153`
     - `185.199.111.153`
   - Or use CNAME: `yourusername.github.io`
4. Commit and push - DNS propagates in 24-48 hours

### Step 4: Set Up AdSense Monetization
1. Apply for Google AdSense at `adsense.google.com`
2. In all HTML files, replace `ca-pub-XXXXXXXXXXXXXXXX` with your Publisher ID
3. Replace `data-ad-slot="XXXXXXXXXX"` with your actual ad slot IDs
4. Test using AdSense Preview Tool before launching

### Step 5: Add Affiliate Links
Find and replace `[AFFILIATE_LINK]` in all HTML files with:
- Crypto exchanges: Binance, Kraken, Coinbase
- Trading platforms: 3Commas, TradingView, Hummingbot
- Product links: Amazon, course platforms, tools
